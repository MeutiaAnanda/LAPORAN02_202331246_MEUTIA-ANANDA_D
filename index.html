1. Mengapa menggunakan CSS dan JavaScript dalam pemrograman web?

CSS diperkenalkan pada tahun 1996 yang digunakan sebagai sebuah cara untuk dapat mengendalikan interface atau tampilan dan gaya dari sebuah web. Dengan kata lain, CSS digunakan dalam melakukan formating pada web sehingga meghasilkan web yang lebih responsif, interaktif dan mempunyai berbagai fitur animasi dan lainnya sehingga memudahkan dalam pengembangan web yang lebih modren dan terbaru dengan berbagai bentuk dan pengembangan yang unik

Sedangkan untuk JavaScipt pada awalnya diciptakan oleh Brendan Eich pada tahun 1995. Sebelum dikenal sebagai JavaScript, nama awalnya adalah LiveScript dengan alasan penggunaan efisien pupularitas bahasa pemorgraman java. Alasan mengapa JavaScript digunakan pada pemograman sebuah web adalah untuk menciptakan sebuah web yang lebih interaktif dengan dilengkapi fitur pendukung seperti efek animasi, validasi formulir, menciptakan konten yang dinamis dan berbagai fitur pendukung lainnya

2. Bagaimana CSS dan JavaScript dapat diintegrasikan dalam pemrograman web?

2 Bagaimana CSS dan JavaSkript dapat diintegrasikan dalam pemograman web? CSS dapat diintegrasikan kedalam sebuah pemograman web dengan cara memasukkan CSS yang telah terintegrasi dengan DOM kedalam halaman yang berisikan HTML. JavaScript dapat diintegrasikan kedalam pemograman sebuah web melalui dua cara, yaitu:

  1. Menambahkan tag JavaScript kedalam sebuah halaman dengan tag<script>Secara langsung.
  2. Menggunakan file JavaScript eksternal dengan tujuan untuk dapat digunakan secara berulang kali serta memudahkan dalam melakukan analisa pemecahan masalah pada pemograman. JavaScript eksternal dapat ditambahkan melalui atribut seperti: <script type+”text/javascript”Src=”ekternalfile.js”><script>

3. Jelaskan apa saja keuntungan menggunakan JavaScript dan CSS?

Keuntungan penggunaan CSS dalam sebuah pemograman web dapat dilihat dengan cara saat ingin mengubah sebuah objek atau element pada sebuah pemograman web biasanya harus dilakukan satu persatu dan berulang kali sehingga akan menghabiskan banyak waktu dan melelahkan. Namun saat CSS diterapkan, kita hanya perlu menambahkan sebuah gaya tunggal pada sebuah element atau beberapa element sekaligus, hal itu juga berlaku untuk melakukan setting pada layout, tampilan dan bentuk dari sebuah halaman web. JavaScript memiliki keuntungan dimana pemogram dapat menciptakan sebuah tampilan web yang interaktif dan kreatif berdasarkan preferensi mereka serta dapat menambahkan berbagai fitur lainnya yang dapat diakses oleh pengguna lainnya, selain itu JavaScript juga memungkinkan klien untuk dapat menghemat waktu dari masalah penghubungan jaringan server yang relatif lama

Ini adalah link ke file CSS Bootstrap eksternal. Dengan menggunakannya, kita dapat memanfaatkan berbagai gaya dan komponen yang Bootstrap sediakan untuk tata letak dan desain yang responsif tanpa perlu menulis CSS khusus.

  1. container: Mengatur margin otomatis dan membuat lebar elemen responsif, sehingga tampilan lebih teratur di berbagai ukuran layar.p-5 dan my-5: Menambahkan jarak padding dalam dan margin vertikal untuk membuat elemen tidak saling menempel.

  4. btn dan btn-primary: Membuat tombol dengan gaya konsisten dan warna utama Bootstrap.
  5. form-floating: Menampilkan label input form di atas saat aktif, membuat form lebih modern.

Bootstrap membantu membuat halaman lebih responsif, terstruktur, dan menarik tanpa perlu banyak kode CSS tambahan.
